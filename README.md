# Simple messenger

## Version 1.0.0

## Features

- Registration and authorization
- Channels
- Chats


## Technology stack:

- React js
- MobX
- Material UI
- Typescript
- React-hook-form
- Socket.io-client

## Install

- 1 Install all packages:

```bash
npm i
# or
yarn install
```

- 2 Start application:

```bash
npm run start
# or
yarn start
```

!!! To work properly, you need to raise the backend.
The backend version of this project [here](https://github.com/pavel-developer2001/simple-messenger-backend).
